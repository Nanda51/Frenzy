# My Discord Bot

This is a simple Discord bot hosted on Railway.

## Features
- Add cards to subthemes.
- List cards by subtheme.
- Find cards in subthemes.
- Bulk add cards.
- Remove cards from subthemes.
- List all cards for a user.

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the bot: `python main.py`
